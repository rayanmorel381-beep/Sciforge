# Periodic Table of Elements

## Overview

The periodic table organizes chemical elements by atomic number, electron configuration, and recurring chemical properties.

> The modern periodic table was first proposed by Dmitri Mendeleev in 1869.

## Electron Configuration

```
Period 1: 1s
Period 2: 2s 2p
Period 3: 3s 3p
Period 4: 4s 3d 4p
Period 5: 5s 4d 5p
Period 6: 6s 4f 5d 6p
Period 7: 7s 5f 6d 7p
```

## Notes

- Hydrogen is unique: it can behave as both a metal and a nonmetal
- Helium has the highest ionization energy of all elements
- Carbon forms more compounds than any other element
